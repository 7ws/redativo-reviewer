Apresentar uma proposta de intervenção para o problema abordado que
respeite os direitos humanos. Propor uma intervenção para o problema
apresentado pelo tema significa sugerir uma iniciativa que busque, mesmo
que minimamente, enfrentá-lo. A elaboração de uma proposta de intervenção
na prova de redação do Enem representa uma ocasião para que o candidato
demonstre o preparo para o exercício da cidadania, para atuar na realidade
em consonância com os direitos humanos.
